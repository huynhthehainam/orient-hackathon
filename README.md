# 1. AI-Powered UI/UX Design System Generator
This project provides an AI-driven system that automatically generates a complete UI/UX design system based on user requirements.
Its goal is to streamline design decisions (layout, style, colors, typography) and ensure consistency, usability, and best practices.
It uses a reasoning engine that:
- Matches product type with UI patterns and styles
- Selects suitable color palettes and typography
- Filters out anti-patterns and applies industry-specific rules

Outputs a full design system including:
- Layout structure (sections, CTA placement)
- Visual style and effects
- Color scheme and fonts
- UX guidelines and pre-delivery checklist
Used to accelerate UI/UX design and generate production-ready interfaces with consistent quality.

# 2. Improving AI Code Quality with Karpathy-Inspired Guidelines
Provides a CLAUDE.md guideline file to improve how AI generates code.
Its goal is to reduce incorrect assumptions, avoid overcomplicated code, and prevent unnecessary changes.
It is built on four key principles:
- Think Before Coding – clarify assumptions and ask questions
- Simplicity First – write the simplest solution possible
- Surgical Changes – modify only what is necessary
- Goal-Driven Execution – work toward verifiable outcomes
